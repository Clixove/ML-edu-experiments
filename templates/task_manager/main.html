Organize modeling steps

Difficulty
  • Operate different pre-processing for different models
  • Search for normalizers and transformers previously generated
  • Migrate some of steps to new tasks
Solution
  • Embed normalizers and transformers inside steps
  • Manage files in a unified space
  • Link steps head-to-tail (support branches)

Recommend best models

Difficulty
  • Draw specialized statistics figures
  • Search for best hyper-parameters and use best enhancing features
  • Compare with different models
Solution
  • Demonstrate features with statistics meaning
  • Automatically figure out evaluation metrics
  • Best models recommended by our data scientist

Internet-oriented applications

Difficulty
  • Work on portable or mobile devices
  • Get a high-performance computer for development
  • Share the ideas with cooperators
  • Repeatedly running models everyday
Solution
  • Build a website and run our software
  • Distribute servers to users center of organizations and enterprises
